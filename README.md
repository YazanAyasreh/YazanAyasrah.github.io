# Yazan Ayasrah - Developer Portfolio

A modern, responsive portfolio website showcasing web development projects and expertise.

## Overview

This is a static portfolio website built with pure HTML5, CSS3, and vanilla JavaScript. It features a dark theme with neon accents, smooth animations, and a clean professional design.

## Features

- **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- **Dark Theme** - Modern dark color scheme with cyan neon accents
- **Smooth Animations** - CSS transitions and keyframe animations
- **Project Showcase** - Featured projects with tech stack and features
- **Contact Information** - Social media links and availability
- **Developer Chat** - Real-time chat with Google OAuth authentication

## Structure

- `index.html` - Homepage with hero, expertise, and about sections
- `projects.html` - Featured projects showcase
- `contact.html` - Contact information and social links
- `chat.html` - Real-time developer chat (requires backend)
- `styles.css` - Main stylesheet

## Developer Chat

The website includes a fully-featured real-time chat system for developers:

- Google OAuth authentication
- Real-time messaging via WebSockets
- Multiple chat rooms
- Online users indicator
- Typing indicators
- Message history

**Setup:** See `README_CHAT.md` for complete backend setup instructions.

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (ES6+)
- Node.js (for chat backend)
- Express.js
- Socket.IO
- PostgreSQL
- Passport.js (Google OAuth)
